# Suyash Nande — Portfolio

Premium, case-study-led engineering portfolio built with Next.js + Tailwind.

## Local development

```sh
npm run dev
```

Open `http://localhost:3000`.

## Environment

- `NEXT_PUBLIC_SITE_URL` (optional): canonical site URL used for `metadataBase` in SEO.

## Notes

- Resume PDF is served from `public/resume.pdf`.
